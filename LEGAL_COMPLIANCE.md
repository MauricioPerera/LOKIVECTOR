### What Constitutes a Violation

1. **Using Commercial Features without license**
   - Using replication in production without Commercial License
   - Using advanced features in commercial products

2. **SaaS without permission**
   - Offering LokiVector as a hosted database service
   - Creating "LokiVector as a Service" products

3. **Trademark misuse**
   - Using "LokiVector" as product name
   - Creating confusing domain names
   - Suggesting official partnership without authorization

4. **License file removal**
   - Distributing without LICENSE file
   - Removing copyright notices
   - Modifying license terms
